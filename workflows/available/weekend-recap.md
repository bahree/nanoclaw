# Weekend Recap

On Monday morning, review what happened over the weekend and set up the week.

1. **Weekend activity** - Summarize any conversations or requests from Saturday and Sunday
2. **Unresolved items** - List anything that was asked but not fully addressed
3. **Week ahead** - If there are any known upcoming events or deadlines, mention them
4. **Suggested actions** - 2-3 things to tackle first today

Keep it brief. The goal is a quick catch-up, not a detailed report.
