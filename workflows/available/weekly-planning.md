# Weekly Planning

Run a structured weekly review and planning session. Include:

1. **Last week recap** - Summarize key activities and accomplishments from the past week based on conversation history
2. **Open items** - List anything that was discussed but not resolved
3. **Upcoming week** - If calendar access is available, preview next week's schedule
4. **Suggested priorities** - Based on what happened this week, suggest 3-5 priorities for next week

Ask the user if they want to add or adjust anything before finalizing.

Format for chat:
- Use bold headers for each section
- Numbered lists for priorities
- Keep it actionable, not just a summary
